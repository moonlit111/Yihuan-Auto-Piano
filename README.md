# NTE_Piano_player

**Neverness to Everness** (异环) 钢琴自动演奏器，支持 MIDI 文件后台自动弹奏。

## 功能特性

- 后台按键模拟（Win32 PostMessage），游戏窗口无需置顶
- 支持加载任意 `.mid` / `.midi` 文件
- 多轨道选择，可单独/组合播放
- 速度调节（0.25x ~ 5.0x）
- 八度偏移（-2 ~ +2），适配不同音域
- 延迟播放，方便切换到游戏窗口
- 内置曲库，首次启动自动下载，支持在线更新
- 自动检测版本更新
- 配置保存/加载（速度、八度、轨道选择）

## 使用方法

1. **以管理员身份运行** `NTE_Piano_player.exe`
2. 点击 **「浏览」** 选择 MIDI 文件，点击 **「加载」**
3. 在 **「目标窗口」** 下拉列表中选择游戏窗口（程序会自动检测异环窗口）
4. 调整速度和八度偏移，点击 **「应用」**
5. 点击 **「▶ 播放」** 开始弹奏

### 快捷键

| 按键 | 功能 |

| F5   | 播放 |
| F6   | 暂停/继续 |

## 常见问题

**Q: 窗口列表中没有游戏窗口？**
点击「刷新」，确保游戏窗口处于打开状态。

**Q: 播放后游戏没有反应？**
- 确认选择了正确的游戏窗口（不是启动器）
- 确认游戏内钢琴界面已打开
- 确认以管理员身份运行本程序

**Q: 很多音符没有播放？**
点击八度偏移旁的「应用」按钮，查看映射范围内的音符数量。如果比例过低，调整八度偏移后重新应用。

**Q: 播放速度不对？**
使用速度滑块调节，MIDI 文件的 BPM 可能与游戏不匹配，建议从 0.8x 开始微调。
