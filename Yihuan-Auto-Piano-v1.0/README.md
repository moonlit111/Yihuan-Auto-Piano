# Yihuan Auto Piano - 异环钢琴自动演奏器

**异环 (Neverness to Everness)** 自动钢琴演奏工具，支持 MIDI 文件自动弹奏。

## 功能

- 加载 MIDI 文件，自动解析音符并模拟键盘弹奏
- 多轨道选择（复选框），支持任意轨道组合
- 播放速度调节（0.1x - 5.0x）
- 八度偏移（-2 ~ +2）
- 后台弹奏（窗口不在前台也能演奏）
- 全局快捷键：Ctrl+Alt+P 播放 / Ctrl+Alt+O 暂停/继续 / Ctrl+Alt+E 停止
- 每首 MIDI 可保存配置文件，下次自动加载
- 导出修改后的 MIDI 文件
- 自动检测异环窗口，无需手动选择
- 内置 25 首示例 MIDI 歌曲

## 使用方法

1. 以管理员身份运行 `MidiPianoPlayer.exe`
2. 点击「浏览」选择 MIDI 文件（默认打开 `default_songs` 文件夹）
3. 选择要播放的轨道
4. 调整速度和八度偏移
5. 在异环中打开钢琴界面
6. 点击「播放」或按 `Ctrl+Alt+P` 开始弹奏

## 注意事项

- **必须以管理员身份运行**，否则按键可能无法发送到异环窗口
- 异环需要打开钢琴界面
- 支持的音域：C3 - B5（3 个八度，36 键）
